Job Search Web Application

For this project, a group of peers and I created a job search web application using third party APIs. I was responsible for integrating the Github Jobs API for the job search, creating search bars for description and location, and dynamically generating cards to display job search results fetched from the API.

Note: As of February 2021, there is a CORS issue with the API used. I am working on a fix to this error. Until then, the site can be used by following the cors-anywhere link in the console after attempting a job search, then clicking "Request temporary access to the demo server".
